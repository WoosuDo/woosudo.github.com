利用Sublime的Snippet实现代码的自动完成

  • Sublime
  • web开发
笔记

什么是Snippet？

Snippet在英文中是片段的意思。Sublime提供Snippet功能，让我们可以自定义自己常用的代码片段，然后通过特定的字符组合佐以Tab键完成自动补完。

在前端的教学视频里经常能见到Snippet的调用。例如输入：

xt

再按Tab键，Sublime就会自动补完以下代码：

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title></title>
</head>

<body>
</body>
</html>

Sublime内置了一些简单的Snippet，不过数量稀少。而像上述HTML的DOCTYPE等信息的补完，本可通过鼎鼎大名的 Zen Coding（Emmet）插件来实现，可惜的是当前版本的Zen似乎并不支持这一功能，所以我们只好自己动手。

如何使用Snippet

Snippet的使用非常简单。

用Shift + Command + P呼出命令行，输入snippet，即可显示现有的Snippet。

列表右侧显示了调用对应Snippet的快捷键。如上图第一行的Snippet:html，快捷键为html,tab，即是说，只要在sublime中输入html，再按下Tab键，即可调用对应的Snippet。

用户也可以通过Tools-Snippet来使用Snippet。

如何设置自己的Snippet

在Tools菜单中选择New Snippet，出现如下代码：

<snippet>
    <content><![CDATA[
        Hello, ${1:this} is a ${2:snippet}.
    ]]></content>
    <!-- Optional: Set a tabTrigger to define how to trigger the snippet -->
    <!-- <tabTrigger>hello</tabTrigger> -->
    <!-- Optional: Set a scope to limit where the snippet will trigger -->
    <!-- <scope>source.python</scope> -->
</snippet>

CDATA标签内的

Hello, ${1:this} is a ${2:snippet}.

即为代码片段。你可以用自己的代码片段代替它。

tabTrigger标签定义了调用Snippet的快捷键，scope标签则定义了该Snippet的有效区域。上图中

source.python

表示该Snippet在以.python为扩展名的文件中生效。若想其在html文件中生效，则scope标签内应为

text.html

以下是文章最初所说输入xt + Tab自动生成html DOCTYPE等信息的Snippet：

<snippet>
    <content><![CDATA[
    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
    <html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title></title>
    </head>

    <body>
    </body>
    </html>
    ]]></content>
    <!-- Optional: Set a tabTrigger to define how to trigger the snippet -->
    <tabTrigger>xt</tabTrigger>
    <!-- Optional: Set a scope to limit where the snippet will trigger -->
    <!-- <scope>source.python</scope> -->
</snippet>

将该文件存入默认路径，命名为htmlXT.sublime-snippet。文件名可随意设置，但扩展名必须为.sublime-snippet。

重启Sublime，刚刚设置的Snippet即可正常使用。
